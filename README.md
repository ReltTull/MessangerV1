# MessangerV1

Идея следующая: в качестве сервера здесь файл .txt для текстовых сообщений и папка - для файловых. Э (экземпляр) Chat может создать Э TextMessage, а Э подклассов User 
может наполнить Э TextMessage. Далее Э Chat может записать созданное сообщение в текстовый файл MessagesStory. Connector устанавливает путь к хранилищу(MessagesStory). 
Возможность отправки файлов пока не реализовал. Могу указать файл для отправки, но пока не могу понять, как переместить в папку для файловых сообщений, что выполняет 
функцию отправки файлового сообщения.
